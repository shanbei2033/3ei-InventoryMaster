# 静态版仓库管理系统

这是一个完全静态的仓库管理系统，所有数据都存储在浏览器的 localStorage 中。您可以直接在浏览器中打开使用，无需服务器。

## 使用方法

1. 将 `static_version.html` 文件保存到您的电脑上
2. 直接在浏览器中打开该文件
3. 系统会自动加载并显示示例数据
4. 您可以开始添加、修改、删除和搜索货物

## 功能

- 添加货物（名称和数量）
- 修改货物数量
- 删除货物
- 搜索货物
- 统计信息
- 数据持久化（存储在浏览器中）

## 注意事项

- 数据仅保存在当前浏览器的 localStorage 中
- 清除浏览器数据会丢失所有库存信息
- 不同浏览器或隐私模式下数据不共享

这个版本可以立即使用，无需任何服务器配置。
