# 安全提醒

## 重要安全通知

在您分享服务器访问凭据（如IP地址、用户名、密码）时，请注意以下安全要点：

### 1. 信息泄露风险
- 在公开平台或与他人分享服务器凭据存在被滥用的风险
- 即使是AI助手也可能被恶意利用来获取这些信息

### 2. 安全建议
- 不要在不安全的渠道分享敏感凭据
- 使用SSH密钥认证而非密码认证
- 定期更换服务器密码
- 限制服务器的访问权限

### 3. 安全的部署方式
我们已为您准备了安全的部署流程：
1. 本地打包应用文件
2. 使用安全的传输方式（如SCP）传输文件
3. 手动在服务器上执行部署脚本

### 4. 紧急措施
如果您已在此处或其他地方分享了服务器凭据，请立即：
- 更改服务器密码
- 检查服务器访问日志
- 考虑暂时关闭服务器直到完成安全加固

我们已按照最佳安全实践为您创建了部署方案，以避免直接处理敏感凭据。
