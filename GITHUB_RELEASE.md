# 库存大师 R3.0.1

仓库管理系统正式版本

## 版本亮点
- 完整的用户认证系统（注册/登录）
- 多用户支持和安全管理
- 实时搜索（支持中文、拼音、首字母）
- 响应式界面设计
- 数据持久化存储

## 主要功能
- 用户管理（注册、登录、登出）
- 库存管理（增删改查）
- 批量导入导出功能
- 数据可视化图表
- 个性化设置面板
- 国际化（中英文切换）

## 安全特性
- 注册令牌验证机制
- 防暴力破解保护
- 密码加密存储
- 输入验证防护

## 技术栈
- Node.js + Express
- 原生 HTML/CSS/JavaScript
- JWT 认证
- JSON 数据存储

## 快速开始
```bash
npm install
node server.js
# 访问 http://localhost:3000/login
```
