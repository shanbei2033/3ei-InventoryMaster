# 仓库管理系统 (InventoryMaster)

一个功能丰富的库存管理Web应用程序，具有实时数据同步、图表可视化和多平台访问能力。

## 主要功能

- 添加、编辑、删除库存项目
- 实时库存数量跟踪
- 低库存预警系统
- 数据导入导出功能
- 交互式图表展示库存趋势
- 移动端适配
- 离线支持（PWA）
- 搜索功能
- 批量操作
- 展开/收回的操作面板
- 全选功能

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- Express.js (后端服务器)
- Chart.js (图表库)
- Node.js

## 快速开始

1. 克隆此仓库
2. 安装依赖：`npm install`
3. 启动服务器：`node server.js`
4. 访问 `http://localhost:3000`

## 部署

系统支持多种部署方式：
- 本地部署
- WSL部署
- Android APK打包
- 静态版本部署

详细部署指南请参见文档文件。
