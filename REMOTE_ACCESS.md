# 仓库管理系统 - 远程访问指南

## 端口转发配置

由于您在路由器上挂了梯子（代理服务），您可以通过以下方式从外部访问仓库管理系统：

### 方法 1: 路由器端口转发
1. 登录您的路由器管理界面
2. 找到"端口转发"或"NAT转发"设置
3. 添加一条新规则：
   - 外部端口：任选一个（如 8080）
   - 内部IP地址：172.26.11.73
   - 内部端口：3000
4. 保存设置
5. 之后可以从外部通过 "您的公网IP:8080" 访问

### 方法 2: 使用您的代理服务
如果您使用的代理服务支持TCP端口转发，请参考其文档配置将外部端口映射到内网的 3000 端口。

### 方法 3: 注册 ngrok 账户（推荐）
1. 访问 https://dashboard.ngrok.com/signup 注册免费账户
2. 获取您的 authtoken
3. 运行以下命令：
   ```bash
   ngrok config add-authtoken YOUR_AUTHTOKEN_HERE
   cd ~/warehouse-manager && npx ngrok http 3000
   ```

### 方法 4: 重新启动本地服务
确保本地服务正在运行：
```bash
cd ~/warehouse-manager
# 检查服务是否运行
ps aux | grep server.js
# 如果没有运行，则启动
node server.js &
```

## 当前本地访问地址
- http://172.26.11.73:3000 （局域网内访问）

## 注意事项
- 确保防火墙允许端口 3000 的访问
- 服务器必须保持运行状态
- 如果服务器重启，需要重新启动 node server.js
