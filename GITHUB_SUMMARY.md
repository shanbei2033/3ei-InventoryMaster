# 库存大师 (InventoryMaster) - 仓库管理系统

专业库存管理解决方案，支持多用户、安全认证和实时搜索功能。

## 主要特性

- ✅ **多用户支持** - 安全的用户认证系统
- ✅ **实时搜索** - 支持中文、拼音及首字母搜索
- ✅ **数据持久化** - JSON文件存储
- ✅ **响应式设计** - 适配桌面端和移动端
- ✅ **国际化** - 支持中英文界面切换
- ✅ **安全防护** - 防暴力破解、输入验证

## 技术栈

- **后端**: Node.js + Express
- **前端**: HTML/CSS/JavaScript
- **认证**: JWT + bcrypt
- **搜索**: 拼音匹配算法

## 快速开始

```bash
# 安装依赖
npm install

# 启动服务器
node server.js

# 访问 http://localhost:3000/login
```
