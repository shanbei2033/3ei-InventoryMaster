# 库存大师 (InventoryMaster) R3.0.1 发布说明

## 🎉 版本亮点

库存大师 R3.0.1 是一个功能完备的库存管理系统，基于OpenClaw平台由AI小扇和小贝开发。此版本包含了完整的用户认证系统、现代化的用户界面和强大的库存管理功能。

## ✨ 新增功能

### 核心功能
- **多用户支持** - 支持多个用户同时使用系统
- **安全认证** - 完整的用户注册和登录系统，使用JWT进行会话管理
- **实时搜索** - 支持中文、拼音及首字母搜索功能
- **数据持久化** - 自动保存库存数据到本地文件

### 界面功能
- **国际化支持** - 支持中英文界面切换
- **响应式设计** - 适配桌面端和移动设备
- **用户管理** - 用户资料和退出功能
- **设置面板** - 个性化配置选项

### 管理功能
- **库存管理** - 添加、编辑、删除、搜索商品
- **批量操作** - 支持导入导出Excel格式数据
- **统计图表** - 可视化库存数据分析
- **全选功能** - 便捷的批量选择操作

## 🔒 安全特性

- **注册令牌** - 需要管理员生成的令牌才能注册新用户
- **防暴力破解** - IP封禁机制防止恶意登录尝试
- **输入验证** - 防止XSS和注入攻击
- **密码加密** - 使用bcrypt安全存储用户密码
- **会话管理** - JWT令牌管理用户会话，具备自动过期功能

## 🛠️ 技术改进

- **完整重构** - R3.0.1版本对前端JavaScript进行了完整重构
- **服务器优先策略** - 实现服务器优先的数据同步策略
- **性能优化** - 修复删除、图表显示、全选等功能问题
- **错误处理** - 增强错误处理和用户体验

## 📋 技术栈

- **后端**: Node.js + Express
- **前端**: 原生HTML/CSS/JavaScript
- **认证**: JWT + bcrypt
- **数据库**: JSON文件存储
- **搜索**: 拼音和首字母匹配算法

## 🚀 快速开始

### 环境要求
- Node.js v14.0 或更高版本
- npm (随Node.js一起安装)

### 安装步骤
1. 克隆仓库到本地
2. 进入项目目录，安装依赖：
   ```bash
   npm install
   ```
3. 启动服务器：
   ```bash
   node server.js
   ```
4. 打开浏览器访问 `http://localhost:3000/login`

### 首次使用
1. 生成注册令牌：`node generate-token.js`
2. 访问 `http://localhost:3000/register` 注册账户
3. 使用注册的账号登录系统
