# 仓库管理系统 - 移动端访问指南

## 通过手机浏览器访问

由于直接打包成 APK 需要复杂的 Android 开发环境，我们采用了 PWA (Progressive Web App) 方案，您可以通过以下步骤在手机上使用仓库管理系统：

### 步骤 1: 确认服务器运行
确保服务器仍在运行 (端口 3000)：
```bash
cd ~/warehouse-manager && node server.js &
```

### 步骤 2: 创建公网访问地址
由于您没有静态公网IP，我们使用 localtunnel 创建临时公网访问地址：
```bash
cd ~/warehouse-manager && npx localtunnel --port 3000
```

### 步骤 3: 在手机浏览器访问
在手机浏览器中访问 localtunnel 提供的公网地址，例如：
- https://slimy-worms-shout.loca.lt (这只是示例，实际地址会不同)

### 步骤 4: 添加到主屏幕 (可选)
在手机浏览器中打开页面后，您可以：

#### 对于 Android (Chrome):
1. 点击浏览器菜单 (通常是右上角的三个点)
2. 选择 "添加到主屏幕"
3. 确认添加

#### 对于 iOS (Safari):
1. 点击分享按钮
2. 选择 "添加到主屏幕"
3. 确认添加

### 功能说明
- 添加货物：输入名称和数量后点击"添加货物"
- 修改数量：使用 +/- 按钮调整数量
- 删除货物：点击"删除"按钮
- 搜索货物：在搜索框中输入关键词
- 数据持久化：所有数据都保存在本地存储中

### 注意事项
- localtunnel 提供的地址是临时的，每次重启服务都会改变
- 服务器和 localtunnel 都需要保持运行状态
- 数据保存在浏览器本地存储中，清除浏览器数据会导致数据丢失
- localtunnel 服务可能会有连接数限制

## 重新启动服务器
如果服务器停止运行，可以使用以下命令重新启动：
```bash
cd ~/warehouse-manager && node server.js &
```
